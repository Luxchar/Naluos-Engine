    The Naluos toolkit is an easy to learn, free and open source, platform for building graphical applications for desktop, mobile and beyond. Combining the power and simplicity of the Python programming language with a carefully crafted library of widgets it is now easier than ever before to build your application and deploy it across all platforms and stores.
