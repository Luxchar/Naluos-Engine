    The Naluos toolkit is an easy to learn, free and open source, platform for building graphical applications for desktop, mobile and beyond. Combining the power and simplicity of the Python programming language with a carefully crafted library of widgets it is now easier than ever before to build your application and deploy it across all platforms and stores.

    A new hope!

    What happens to us creating Naluos ? Our ambition and goal was to create our own Game Engine on Python. The ones that offered on the market did not suit us, so we decided to undertake the configuration and creation of this one.

    Have fun !

  • Naluos is a free cross-platform library that facilitates the development of real-time video games with the Js programming language.

    Naluos makes it possible to program the multimedia part (graphics, sound and keyboard, mouse or joystick inputs), without coming up against the difficulties of basic languages ​​such as C and its derivatives. This is based on the assumption that the multimedia part, often the most constrained to program in such a game, is sufficiently independent of the logic of the game itself so that a high-level language can be used for the structure of the game.

    We can also notice that Naluos is no longer used exclusively for video games, but also for various applications requiring graphics.
