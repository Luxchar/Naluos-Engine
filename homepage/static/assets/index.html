    Naluos Engine is an easy to learn, free and open source platform to build graphical applications for desktop, mobile and beyond. Thanks to this javascript framework it is now easier than ever before to build your application and deploy it across all platforms !

    A new hope!

    What happens to us creating Naluos ? Our ambition and goal was to create our own Game Engine on Javascript. other frameworks did not suit us because we thought we could simplify the process of making a game, so we decided to undertake the creation of this one.

    Have fun !

  • Naluos is a free cross-platform library that facilitates the development of real-time video games with the Javascript programming language.

    Naluos makes it possible to program the multimedia part (graphics, sounds and keyboard, mouse or joystick inputs), without coming up against the difficulties of collision or animations.

    Naluos is no longer used exclusively for video games, but also for various applications requiring graphics !
